                        Diet-Survey
			===========

What is Diet-Survey?
-------------------

Diet-Survey is a web application for calculating nutrient intake of a
family based on their food consumption in last 24 hours.

Diet-Survey is written in PHP and Javascript for server installation.

Downloading
-----------

Latest release of Diet-Survey can be downloaded from http://code.agnibho.com/dietsurvey/

The source code is available at https://github.com/agnibho/dietsurvey/

Installation
------------

Please see the file named INSTALL.txt for installation instructions.

Usage
-----

After installation Diet-Survey can be accessed from the designated
URL.

You need to enter some health information about the family members and
the list and amount of food they have consumed in the last 24 hours.

A hosted version of Diet-Survey is available at
http://dietsurvey.agnibho.com/
